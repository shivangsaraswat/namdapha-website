### 1. Google Search Console (MOST CRITICAL)
**Link:** https://search.google.com/search-console

**Steps:**
1. Click "Add Property"
2. Enter: `namdapha.iitmbs.org`
3. Choose "HTML tag" verification method
4. Copy the verification code (looks like: `google-site-verification=ABC123XYZ`)
5. Replace `YOUR_VERIFICATION_CODE_HERE` in `frontend/app/layout.tsx` with your code
6. Deploy the change
7. Click "Verify" in Search Console
8. Go to "Sitemaps" ‚Üí Add: `https://namdapha.iitmbs.org/sitemap.xml`
9. Go to "URL Inspection" ‚Üí Request indexing for:
   - https://namdapha.iitmbs.org
   - https://namdapha.iitmbs.org/about
   - https://namdapha.iitmbs.org/council
   - https://namdapha.iitmbs.org/teams
   - https://namdapha.iitmbs.org/events
   - https://namdapha.iitmbs.org/resources

**Timeline:** Indexing starts in 1-3 days
